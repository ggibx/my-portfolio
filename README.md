# 简历上传页面

一个使用 Vue 3 构建的简历 PDF 上传和预览应用。

## 功能特性

- 📄 PDF 文件上传（点击或拖拽）
- 👀 PDF 在线预览
- 📥 下载已上传的简历
- 🎨 现代化的 UI 设计
- 📱 响应式布局，支持移动端

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

## 使用方法

1. 打开浏览器访问 `http://localhost:3000`
2. 点击上传区域或直接拖拽 PDF 文件
3. 上传后可以预览和下载简历

## 技术栈

- Vue 3
- Vite
- 原生 JavaScript（无需额外依赖）

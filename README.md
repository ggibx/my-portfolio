# 简历网站

基于Vue 3构建的简历网站，仿照PDF样式设计。

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run dev
```

项目将在 http://localhost:3000 启动

## 构建生产版本

```bash
npm run build
```

## 预览构建结果

```bash
npm run preview
```

## 功能特点

- 📄 PDF样式的简历布局
- 🖨️ 打印友好的样式设计
- 📱 响应式设计，支持移动端
- 🎨 专业的排版和样式

## 技术栈

- Vue 3
- Vite
- CSS3
